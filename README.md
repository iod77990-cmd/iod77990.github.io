# 📚 지식 저장소 (Knowledge Base)

배운 내용을 잊지 않기 위해, 그리고 논리적 사고를 정리하기 위해 기록하는 공간입니다. 
주로 **Java**, **SQL**, **CS 기초** 지식을 다룹니다.

## 🛠 Tech Stack
- **Language:** Java, SQL, HTML, JavaScript, CSS
- **Tools:** MacBook, iPad, Obsidian, IntelliJ IDEA 2026.1 IDE, SQLyog
- **Platform:** GitHub Pages, Velog

## 📂 Repository Structure
- `/Java`: 자바 문법 및 문제 풀이, 논리적 실수 기록
- `/SQL`: 데이터베이스 설계 및 쿼리 연습 ([ArticleJDBC 프로젝트 포함])
- `/CsStudy`: 컴퓨터 과학 기초 이론 (운영체제, 네트워크 등)
- `/정보처리기사 오답 개념 정리` : 국가자격증 정보처리기사 기출문제 개념 오답 정리

## ✍️ Principles
1. **이유 기록하기:** 단순히 코드를 적는 게 아니라 '왜' 이렇게 작동하는지 기록합니다.
2. **오류 정리:** 발생한 에러와 해결 과정을 상세히 기록하여 같은 실수를 반복하지 않습니다.
3. **문제풀이** 문제풀이를 통한 새로운 개념이나 키워드 익숙해지기
4. **꾸준함:** 매일 학습한 내용을 마크다운으로 구조화하여 정리합니다.
