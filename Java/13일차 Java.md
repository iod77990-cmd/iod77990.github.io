### 객체 생성 인터페이스
```java

키워드 인터페이스( ex>List)<자료형or클래스이름> 선언이름 = new 자료구조<>();

```
Java 키워드
- instance : static이 붙지 않은 모든 변수와 함수
- static : 정적맴버 키워드로 실행이후 객체를 따로 생성하지 않아도 
	static로 선언한 클래스의 함수에 접근해서 사용 가능
- final : 한번 초기화 이후 바꿀 수 없는 상수, 함수, 상속불가
- abstract : 공통점만 모은 추상클래스 구현은 자식클래스에 구현함
- synchronized : 동기화
- transient : 직렬화 제외

list는 클래스들의 공통점을 모은 추상클래스로 인터페이스로도 사용가능
ex) ArrayList, LinkedList

- list 의 주요 함수
  - add() : 리스트에 요소 추가
  - get() : 지정한 인덱스에 위치한 요소 get
  - remove() : 지정한 인덱스의 요소 삭제
  - size() : 리스트에 저장된 요소의 개수 
  - contains() : 리스트에 지정된 요소를 포함하는지 여부
  - indexOf() : 리스트에서 지정된 요소의 첫번째 인덱스 반환, 없으면 -1 반환
  - clear() : 리스트의 모든 요소 제거

### M.V.C 패턴
- Model : 프로젝트의 핵심 부분, 서비스와 리포지터리로 구성
- View : 보여지는 부분, 콘솔에서는 System.out.print(); 함수의 역할
- Controller : 클라이언트(고객)에게 요청을 받아 서비스에게 답을 하는 역할
