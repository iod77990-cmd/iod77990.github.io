### while문의 scope
- while문 밖의 변수
반복문이 끝나도 생성한 변수를 사용하고 싶을 때

- while문 안의 변수
반복이 끝날 때마다 메모리에서 지우고 매번 초기화됨

- while문 안과 밖의 변수
일명 변수 가리기 현상 발생
밖에서 생성한 변수가 안에서 생성한 변수에 의해 가려지는 현상

### SQL 보완
- desc(describe 약어) 데이터베이스의 구조를 보기 위한 키워드
```sql
desc mysql.db; #mysql에 있는 db의 구조 출력
```
